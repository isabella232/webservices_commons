## Add to new service project

```
git clone https://github.com/nextgis/webservices_commons nextgis_common
```
or
```
git submodule add https://github.com/nextgis/webservices_commons nextgis_common
```

Add to INSTALLED_APPS:
```
    'widget_tweaks',
    'django_gravatar',
    'nextgis_common',
```
